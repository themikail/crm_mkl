# **App Name**: Synergize CRM

## Core Features:

- Company and Contact Management: Create, read, update, and delete companies and contacts, associating multiple contacts with one company.
- Sales Pipeline Management: Manage deals in a Kanban-style pipeline with configurable stages, deal value, expected close date, and owner assignment.
- Activity Logging: Log calls, meetings, emails, tasks, and notes related to companies, contacts, and deals, creating a unified timeline.
- Google Calendar Integration: Schedule meetings directly from the CRM, create Google Calendar events, and sync event metadata to the CRM timeline.
- Manual Email Logging: Log inbound or outbound emails as activities, including subject, direction, and timestamp.
- Google Tasks Integration: Create Google Tasks from CRM tasks, sync due dates and completion status, and link tasks to contacts, companies, and deals. This feature will use an LLM to reason about which task lists, from Google, should be proposed as candidate targets for syncing tasks, based on the name of the list, as well as the current CRM org or user
- Dashboard Reporting: Display key metrics such as deals by stage, forecast, and tasks due today, with filters by user and date range.

## Style Guidelines:

- Primary color: Deep sky blue (#4990E2) to convey professionalism and trust.
- Background color: Very light blue (#F2F8FD), providing a clean and modern backdrop.
- Accent color: Teal (#2E8A99), used for interactive elements and highlights to guide user attention.
- Body and headline font: 'Inter', a sans-serif font, for a modern and neutral look in both headlines and body text.
- Use minimalist, professional icons from a consistent set.
- Mobile-first, responsive layout with a clear navigation structure. Use of Cmd/Ctrl + K command palette for quick actions.
- Subtle animations for transitions and loading states to improve user experience.
